## 아이템 48: API 주석에 TSDoc 사용하기

- IDE는 함수가 호출되는 곳에서 함수에 붙어있는 JSDoc 스타일 주석을 툴팁으로 표시한다.
- 타입스크립트의 관점에서는 TSDoc이다.
  - `@param`, `@returns`
  - 타입 정의에 사용
- 마크다운 형식
- JSDoc처럼 타입 정보를 명시하지 않는다. 타입이 있으니깐
