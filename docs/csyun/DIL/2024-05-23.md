## 아이템 26. 타입 추론에 문맥이 어떻게 사용되는지 이해하기

1. 변수를 뽑아서 별도로 선언했을때 오류가 발생한다면 타입선언을 추가해야 함
2. 변수가 정말로 상수라면 as const를 사용해야 함
3. 상수 단언을 사용하면 정의한 곳이 아닌 사용한 곳에서 오류가 발생함

## 아이템 27. 함수형 기법과 라이브러리로 타입 흐름 유지하기
